# CS Learning Hub

A full-featured content management website built with HTML, CSS, and JavaScript.

## Features

✅ **Top Navigation Bar** - 4 sections: Cyber Security, Robotics/Hardware, Roblox Studio, Multi-Media  
✅ **Admin Post System** - Create, view, and delete posts  
✅ **Post Feed** - Displays up to 3 posts per section  
✅ **Post Details** - Tabs for Overview, Code Base, Lessons, Support  
✅ **Responsive Design** - Works on desktop, tablet, and mobile  

## How to Use

1. Open `index.html` in a web browser
2. Click navigation buttons to filter by section
3. Click "Admin Login" to create new posts
4. Click on posts to view details

## Files

- `index.html` - Main page
- `styles.css` - Styling
- `script.js` - Functionality
- `README.md` - Documentation
